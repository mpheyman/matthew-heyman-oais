# OAIS (v0) — Operating Model

## Definition
Human-orchestrated AI Product Operating AI System (OAIS).

## Philosophy
- Humans own direction.
- Agents own execution.
- Convergence > generation.

## Lifecycle (high level)
Idea intake → Research/Discovery → Spec/Decomposition → Alignment gate → Implementation → Convergence review → Deploy → Audit/retro.

## Deliverables
- Structured docs (indexed)
- Repeatable workflows
- Measurable outcomes (quality, token burn, drift)
