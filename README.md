# matthew-heyman-oais

This repository contains the **OAIS (Operating AI System)** as a single folder, `oais/`, so you can drop it into any project in one step.

## Get the OAIS

- **Clone or download:** Copy the **`oais/`** folder from this repo into your project root. That's it.
- **Docs:** See [oais/README.md](oais/README.md) (operator manual) and [oais/INSTALL.md](oais/INSTALL.md) (install steps).

## Quick install

1. Clone this repo or download the ZIP.
2. Copy the `oais/` folder into your project root.
3. Follow [oais/INSTALL.md](oais/INSTALL.md).

No need to rename or restructure: the folder is already named `oais/` and ready to drop in.
